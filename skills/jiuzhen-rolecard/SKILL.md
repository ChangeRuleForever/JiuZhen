## Workflow

1. Classify the change before editing anything.
- Common-layer changes: theme switching, family overview, universal traits/skills, equipment, follower layout, personal info formulas, quick-copy area.
- Family-layer changes: one family's progression, dedicated sheets, family-specific hidden helpers, or family-only macro logic.

2. Prefer source-first edits.
- Raw Word and Excel materials live in `source/families/` and `source/tables/`.
- Workbook templates and generated workbooks live in `workbooks/`.
- Universal family data lives in `references/data/universal_rolecard_data.json` and is regenerated by `scripts/build_universal_rolecard_data.py`.
- Family source extraction lives in `references/data/family_sources.json` and is regenerated by `scripts/extract_family_sources.py`.
- Workbook rebuild logic lives in `scripts/build_universal_workbook.ps1` and `scripts/update_mojia_workbook.ps1`.
- VBA source of truth lives in `references/vba/`, not only inside the `.xlsm`.

3. Touch the workbook directly only when the change is local and not already encoded in bundled scripts.
- For repeatable data changes, update the bundled script or JSON first.
- For macro changes, patch the exported VBA module first, then re-inject via the PowerShell rebuild script.

4. Preserve linked behavior.
- Do not break the theme dropdown at `基本信息!B2` or the family list at `基本信息!A6:A15`.
- Keep helper ranges hidden and formula-backed: `书籍、特性与技能!W:AD`, `装备列表!M:Z`, and the hidden helper zones in `【墨家】物品库`.
- When adding new dropdown options, update both the source data and the validation/helper logic that feeds the visible cells.

5. Update both macro layers when the change crosses concerns.
- Shared theme colors, universal sheet styling, herb-sheet visibility, and common lookups belong to `references/vba/JiuzhenUniversalMacros.bas`.
- 墨家 bag logic, 魁人 validation, save guards, and workbook event behavior belong to `references/vba/JiuzhenMojiaMacros.updated.bas`, `references/vba/ThisWorkbook.cls`, and `references/vba/Sheet3.cls`.

6. Rebuild and verify instead of assuming.
- Run the relevant PowerShell script after changing bundled sources.
- Re-check sheet visibility, formulas, dropdowns, hidden helper areas, and button assignments.
- Keep `workbooks/玖镇新版角色卡.xlsm` as the generated artifact, `workbooks/templates/新版玖镇角色卡_墨家.xlsm` as the workbook template source, and the bundled scripts/VBA/JSON as the maintainable source.

## Routing Notes

- For any request about workbook structure, common formulas, theme switching, universal traits/skills, equipment, followers, or quick-copy output, read `references/common.md`.
- For 墨家 requests, read `references/families/mojia.md` before touching sheets or VBA.
- For 药家 requests, read `references/families/yaojia.md`; this family uses both common tables and a theme-gated dedicated sheet.
- For 乐正家、张家、岳家、林家、易家、曹家、巫家、亓家, read only the corresponding family file. These families currently reuse the common workbook tables rather than dedicated visible worksheets.
